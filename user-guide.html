Lift Planner User Manual

From first launch and onboarding to completing your workout and reviewing progress.

How to use this guide

Follow the same order as the app. Each section maps to a real screen and action flow.

1. First Launch and Onboarding 2. Home and Main Navigation 3. Setup Before First Workout 4. Start a Workout Session 5. During the Workout 6. Finish and Save Workout 7. After Workout and Next Session 8. Pro and Trainer Benefits 9. Common Issues

1. First Launch and Onboarding

Step 1: Welcome screen

Open the app and review key features. If you already have data, use backup restore options on this screen.

Step 2: Create your first profile

  • Tap Get Started.
  • Enter your name.
  • Select gender.
  • Pick an avatar color.
  • Tap Create Profile.
Tip: Lift Planner supports multiple users on one device. Each user has separate workout history and cycle progress.

2. Home and Main Navigation

After onboarding, you land in the main tab interface:

  • Home: active cycle, start or resume workout, quick actions.
  • Exercises: browse exercise library and details.
  • Inventory: manage available equipment and weights.
  • Settings: app preferences, language, and support items.
Tip: Before your first session, complete inventory and cycle setup for accurate suggestions.

3. Setup Before First Workout

Step 1: Configure inventory

Go to Inventory and add your equipment (barbell, dumbbells, machines, bodyweight options). Include realistic increments or fixed weight options. The app already includes starter inventory to help you begin quickly, and Pro unlocks a larger inventory selection.

Step 2: Create a training cycle

Return to Home and tap Create Training Program.

  • Pick a system template, or create a custom cycle.
  • Set cycle duration (typically 4-7 weeks).
  • Review workout days and routine structure.

Step 3: Fill exercise slots

Open cycle editor and assign exercises to all required slots. The workout button is enabled only when the cycle is ready.

4. Start a Workout Session

Step 1: Start from Home

Tap Start Workout. If a session is already active, use Resume Workout.

Step 2: Choose routine

Select today's routine from the routine selector. The app creates a workout session and opens the active workout screen.

Note: The app can resume interrupted sessions, so you can continue later without losing logged data.

5. During the Workout

Workout screen overview

  • Header shows progress and elapsed time.
  • Each exercise is shown as a card that can expand/collapse.
  • You can open exercise details (notes, history, instructions).

Log sets and performance

  • Enter weight and reps for each set.
  • You cannot manually mark a single set as skipped.
  • Remaining sets are treated as skipped only when you finish an exercise early with sets still incomplete.
  • Use exercise feedback flows when prompted.

Understand weekly RIR targets (and why they matter)

Your cycle uses week-specific RIR targets (Reps In Reserve) to control training stress across the block. Some weeks are intentionally harder, while others are lighter to manage fatigue and improve progression quality.

  • Lower RIR means you finish closer to failure (harder week).
  • Higher RIR means you keep more reps in reserve (easier/recovery week).
  • Aim to match the week's RIR target, not just lift the heaviest possible weight.

How to achieve target RIR in real sessions

  • Adjust load up or down so your last reps match the target effort.
  • You can perform a different number of reps than suggested when needed.
  • If suggested targets feel unrealistic, choose weights and reps that fit your actual performance today.
  • Log what you really did; accurate logs are better than forcing incorrect targets.

First time doing an exercise

On first exposure, the app may not have a suggested weight and may default reps to 10. You should set weight and reps yourself based on your current ability, while still trying to match the current week's RIR target.

Crucial: post-exercise feedback teaches the app how the set actually felt and is a key input for next week's weight/rep suggestions. The more honest your feedback, the better your progression recommendations.

Use workout menu actions

  • Finish Workout: complete the session early or when done.
  • Reset Workout: clear current session entries and restart.
  • Abandon Workout: discard the current session.
  • Go to Home: leave the screen without losing an active session.
Important: to skip the rest of a workout, first start the workout session, then open the three-dot menu and tap Finish Workout.
Tip: Use the user switcher in the top bar if multiple people share the same device.

6. Finish and Save Workout

Step 1: Finish workout

When all exercises are done, use Finish Workout. You can also finish early; the app confirms before completion if exercises remain.

Step 2: Review summary

You will see summary metrics such as completed exercises, total sets, duration, and total volume.

Step 3: Tap Done

Tap Done to finalize and return to normal navigation. The session is saved and progress updates in your cycle.

Saved result: your session data becomes part of workout history and future progression calculations.

7. After Workout and Next Session

  • Return to Home to monitor cycle progress and completed workouts.
  • Open Exercises to review notes, history, and instructions per movement.
  • Update Inventory when your available weights change.
  • Adjust settings or switch users as needed.
Best practice: keep inventory and exercise notes up to date for better session quality and progression decisions.

8. Pro and Trainer Benefits

Why upgrade from Free

Upgrading unlocks advanced planning, customization, and user management features that make the app more flexible for long-term training.

Pro tier includes

  • All app features unlocked.
  • Up to 3 user profiles.
  • Custom exercises and custom equipment.
  • Backup import and restore support.
  • Custom training templates.
  • Larger starter inventory than Free.

Trainer tier includes

  • Everything in Pro.
  • Unlimited user profiles.
  • Built for coaches and personal trainers managing multiple trainees.
  • No trainee limits.
Upgrade path: you can start with Pro and later upgrade to Trainer when you need unlimited clients/profiles.

9. Common Issues

  • Start Workout is disabled: add an active cycle and fill all required exercise slots.
  • Wrong user data: switch active user from Home or workout top bar.
  • Lost session screen: go to Home and tap Resume Workout.
  • Need to restart session: open workout menu and use Reset Workout.
